# FT3 Project Feature List

## 🟢 Core Features

### 1. User & Group System
- User signup/login
- Create groups (Trip / Roommates / Event)
- Invite members via link
- Add/remove participants

### 2. Expense Entry System
- Add expenses: Equal, Unequal, Percentage split
- Assign payer
- Add description (food, travel, etc.)
- Optional chat-style input

### 3. Auto Split Engine
- Automatically divides expenses among users
- Updates balances instantly

### 4. Balance Calculation
- Show "who owes whom"
- Display total owed per user
- Real-time updates

### 5. Settlement Optimizer
- Minimize number of transactions
- Generate optimal payment flow
- Display step-by-step settlement

### 6. Visual Debt Graph
- Nodes = users
- Edges = debts

### 7. One-Click Settlement Plan
- Button “Optimize”
- Shows before vs after settlement

## 🟡 Intermediate Features

### 1. Dashboard
- Total group spending
- Individual contributions
- Net balance summary

### 2. Transaction History
- Timeline of expenses
- Filter by date, user, category

### 3. Analytics & Charts
- Category-wise spending (pie chart)
- Monthly trends (bar chart)

### 4. Notifications
- Reminders: "You owe ₹500", "New expense added"

### 5. Settle Up System
- Mark debts as paid
- Track completed transactions

### 6. Expense Categorization
- Food, Travel, Bills, etc.

## 🔵 AI Features

### 1. NLP Expense Input
- Parse user input like "Paid 500 for dinner"
- Extract amount, category, payer

### 2. Spending Insights
- Detect patterns and trends
- Example: "Food spending increased 25%"

### 3. Smart Suggestions
- Recommend best split type
- Suggest ways to reduce unnecessary spending

### 4. AI Chat Assistant
- Answer queries like "Who owes me?" or "Show my expenses"

### 5. Predictive Spending (Optional)
- Forecast future expenses

## 🔗 Real-World Integrations

### 1. Payment Integration (Demo)
- "Pay Now" button
- UPI / Razorpay redirect

### 2. Reminder System
- Payment reminders and due alerts

### 3. Location Tagging
- Tag expense location

## 🔴 Advanced Features

### 1. Multi-Group Optimization
- Combine debts across multiple groups

### 2. Budget System
- Set monthly budget
- Alerts on overspending

### 3. Fraud / Error Detection
- Detect duplicate entries
- Flag suspicious transactions

### 4. Gamification
- Leaderboards: "Top spender", "Most saver"

### 5. Voice Input
- Add expense via voice

### 6. Offline Mode
- Save expenses offline and sync later

## 🟤 Ultra WOW Features

### 1. Blockchain Ledger
- Immutable transaction records
- No disputes

### 2. AI Financial Advisor
- Suggest savings and improved spending habits

### 3. Financial Health Score
- Score users based on spending patterns

## 🎯 Build Strategy

- 🟢 Minimum competitive: Group system, Expense system, Settlement optimizer, Dashboard, Graph
- 🟡 To stand out: NLP input, Charts, Insights
- 🔥 To win: Optimization demo, AI features, Clean UI
